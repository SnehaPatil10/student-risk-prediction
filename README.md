# 🎓 Student Risk Prediction System

---

## 📌 Overview

The Student Risk Prediction System is a Machine Learning-based project designed to identify students who are at risk of poor academic performance.

It analyzes factors like attendance, internal marks, assignment scores, and gender to predict whether a student is **At Risk** or **Not At Risk**.

The system also provides a simple interactive dashboard using Streamlit.

---

## 🎯 Objectives

- Predict student academic risk  
- Analyze student performance data  
- Improve prediction accuracy using Machine Learning  
- Provide an easy-to-use interface  
- Help teachers identify weak students early  

---
